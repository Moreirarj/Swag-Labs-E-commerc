# Swag Labs E-commerce Automation

Projeto de automação E2E desenvolvido com Playwright e TypeScript.

## Tecnologias

* Playwright
* TypeScript
* Page Object Model (POM)
* GitHub

## Cenários Automatizados

### Login

* Login com sucesso
* Usuário bloqueado
* Senha inválida
* Campos obrigatórios

## Como executar

```bash
npm install
npx playwright test
```

## Relatório

```bash
npx playwright show-report
```
